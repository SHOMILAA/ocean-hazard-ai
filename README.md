# 🌊 Ocean Hazard Alert System

## 📌 Project Description
This project is an AI-based web application that detects potential ocean hazards using social media content.

It analyzes both:
- 📝 Text (fake news detection)
- 🖼️ Images (hazard detection)

The system filters misinformation and provides early hazard alerts.

---

## 🚀 Features
- Fake news detection using Machine Learning
- Image-based ocean hazard detection
- Hazard classification (Cyclone, Storm, Flood)
- Alert system with confidence score
- Dashboard to view hazard history
- Multimodal AI (Text + Image analysis)

---

## 🛠️ Technologies Used
- Python (Flask)
- HTML, CSS, JavaScript
- Scikit-learn (ML model)
- PyTorch (Image model)
- SQLite Database

---

## ▶️ How to Run
1. Install dependencies:
pip install -r requirements.txt

2. Run the app:
python app.py

3. Open browser:
http://127.0.0.1:5000
